## 1. はじめに

本作品では，1990 年代のサバイバルホラーゲームに特徴的な「固定視点カメラ」の演出を，VR と実空間を組み合わせることで現実世界に再構築する．固定視点カメラ越しに自分自身の身体を観察するという，ゲーム特有の知覚条件を実世界の身体経験として体験者に与えることで，現代 VR では生起しにくい種類の恐怖と身体性を提示する．

『バイオハザード』『サイレントヒル』『クロックタワー』など，1990 年代後半のサバイバルホラーゲームは，プリレンダ背景に固定カメラを配置する演出によって独特の恐怖を作り上げてきた [1]．プレイヤーは自分のキャラクターを第三者の画角で見続け，画面外から迫る脅威，カメラ切替の瞬間に現れる敵，自分の死角に立つ何者かといった，一人称視点では成立しない恐怖を体験した．しかし 2000 年代以降，ゲームは三人称追従カメラ・一人称視点へと収斂し，固定視点演出は技術的・操作性の理由から廃れていった．

近年の VR 体験はこの流れをさらに推し進め，「自分の目で直接見る」一人称没入を前提として設計されている．一方で，VR における身体所有感に関する研究では，自分の身体を外側から観察する「アウトオブボディ」状況下でも，視覚と体性感覚の同期があれば自己同一視が成立することが示されている [2]．また Slater らは，三人称視点 VR が一人称視点と異なる感情・認知反応を引き出すことを報告している [3]．これらの知見は，あえて第三者の固定画角を強制する VR 体験が，一人称没入とは異質な知覚条件を生み出しうることを示唆する．

本企画《固定視点実世界探索ホラー》では，こうした固定視点演出を VR と実空間に持ち込み，体験者が「他人の目で自分自身を見続けながら，自分の身体で実世界を探索する」という，視覚と身体運動の分離を伴う体験を構築する．体験者は HMD を装着して実空間を自身の足で移動するが，HMD に提示されるのは空間内に設置された複数の固定カメラから取得した自身を含む映像のみである．視覚は他者の画角に固定されたまま，移動と姿勢制御のみが体験者の身体に委ねられる．この分離は，固定カメラの画角外に対する不可知性，第三者視点に常時晒されることによる被観測感，ゾーン間のカメラ切替に伴う方向感覚の不連続性といった，一人称 VR および追従カメラ型ゲームのいずれにおいても生起しない知覚条件を作り出す．

## 2. システム構成

本システムは，体験スペース内に設置した複数の固定カメラ（スマートフォン）が撮影する実世界映像を Wi-Fi 経由で Meta Quest 3 に伝送し，HMD 内の仮想スクリーンに提示することで，体験者に「他人の画角越しの自分自身」を見せる構造を持つ（図 1）．現実の展示空間は HMD 内映像で覆い隠され，体験者の視野からは固定カメラ映像以外の現実の手がかりが排除される．体験者の頭部位置・姿勢は Quest 3 のインサイドアウトトラッキングで取得され，現在位置するゾーンに応じて表示カメラが自動的に切り替わる．

### 2.1 ハードウェア構成

HMD には Meta Quest 3 を用い，6DoF インサイドアウトトラッキングで体験者の位置・姿勢を取得する．固定カメラとしてスマートフォン 3 台を体験スペースの異なる位置に三脚で設置し，DroidCam / IP Webcam による MJPEG over Wi-Fi で映像を配信する．会場提供回線への依存を避けるため，ネットワークはモバイルルーターによる閉じたローカル AP を持ち込む．展示空間中央には腰高（約 1m）の仕切り壁を配置し，3 つのゾーンに分割する．

### 2.2 映像取り込みとレイテンシ設計

体験者の入力（位置・姿勢の変化）から表示映像の更新までのレイテンシは，没入感と恐怖演出のいずれにも影響する．先行研究では，VR 環境下でモーションから表示までの遅延が約 60ms を超えると違和感が知覚され始めることが報告されている [4]．本作の映像経路（カメラ撮影 → Wi-Fi 伝送 → MJPEG デコード → HMD 表示）は撮影系の遅延が支配的であり，本作では全体遅延を 100ms 以内に抑えることを目標とする．Unity 側では MJPEG ストリームをテクスチャに非同期デコードし，フレーム内の GC アロケーションを排除することで Quest 3 の 90Hz 表示を維持する．

### 2.3 映像演出と CG 合成

固定カメラ映像は Shader Graph によるリアルタイムフィルタ（走査線・色収差・ノイズ・ビネット・低解像度化）を経て，1990 年代の監視カメラ風 UI（カメラ番号・タイムスタンプ）と共に提示される．これに加え，実空間には存在しない CG オブジェクト（血痕・落書き・歪んだ人影）を映像上に α 合成することで，「見えるが触れないもの」を作り出し，実世界とフィクションの境界を揺るがす．カメラ切替時には一瞬の暗転とノイズを挿入し，固定視点ゲームのロード演出を踏襲する．

### 2.4 位置駆動の映像インタラクション

体験者の物理的なゾーン移動・姿勢変化（しゃがむ・振り向く）は，アクティブカメラの切替および CG 演出のトリガーとして機能する．「移動するとカメラが切り替わる」という固定視点ゲーム特有の構造を，画面入力ではなく身体運動で発火させる点が本作の技術的新規性である．これにより体験者は，視覚的には他人の画角に縛られながらも，自分の身体運動が映像の構造そのものを書き換えていることを徐々に理解する．

## 3. 体験の流れ

本企画の体験は，固定視点カメラ越しに実空間を歩く中で恐怖が段階的に立ち上がる構成を取る．以下にその具体的な流れを示す（約 3 分）．

1. **導入と HMD 装着**：体験者は Meta Quest 3 を装着し，実空間に対応するパススルーまたはニュートラルな映像で立ち位置を確認する．固定カメラ映像への切り替えと共に体験が開始される．
2. **シーン 1：廊下（カメラ①，立位移動）**：斜め上からの俯瞰画角．体験者は仕切り壁に沿って歩き，奥のゾーンへ進む．映像にはノイズと走査線フィルタが乗り，画面端に「何か」が一瞬映る演出が挿入される．
3. **シーン 2：角を曲がる（カメラ②，立位移動）**：ゾーン境界を跨ぐとカメラが切り替わり，視点が反転する瞬間に方向感覚が一度失われる．仕切り壁の裏側を覗き込む，屈むなどの身体動作によって，映像内に CG オブジェクト（落書き・血痕・人影）が出現する．
4. **シーン 3：部屋（カメラ③，着座）**：椅子に座らされ，正面の固定カメラに自分が大きく映る．背後（実空間にも CG にも）に異変が現れ，体験者は振り返るがカメラは動かないため自分の後頭部しか見えない．画面外への不可知性が最大化される．
5. **クロージング**：暗転とともに体験が終了し，HMD を外す．

体験中はオペレーター 2 名（HMD 装着補助 1 名 + 進行・安全監視 1 名）が常駐し，体験者の死角に立ちアシストする．移動はすべて自分の足によるリアル移動のため VR 酔いは発生しにくいが，固定カメラ視点は通常の VR と感覚が異なるため，希望者には事前に短いプレ体験を提供する．

## 4. むすび

本作品では，1990 年代サバイバルホラーゲームの「固定視点カメラ」演出を，VR と実空間の組み合わせによって現実世界に再構築する体験装置を提案した．体験者は他人の画角越しに自分自身の身体を観察しながら，自分の足で実世界を探索する．視覚と身体運動の分離，画角外への不可知性，被観測感の常態化といった知覚条件は，一人称没入を前提とする現代 VR のいずれにおいても生起しないものであり，固定視点という「失われた表現」を画面の外に解放する試みである．

本企画は単なるホラー作品にとどまらず，「視覚と身体の乖離が体験者にもたらす情動・認知変化」を実装と展示を通じて検討する，VR の視点設計に関するひとつの実験でもある．既存の一人称没入型 VR と並ぶ選択肢として，第三者の固定画角を採用する設計可能性を提示することに貢献したい．
