# 撮影納品仕様（あなた → シュビー）

あなたは**生の映像クリップを撮って渡すだけ**。CCTV 風 UI（カメラ番号・タイムスタンプ・REC・四隅ブラケット）・**字幕**・「実機/イメージ映像」表記・尺合わせ・並べ・走査線質感は**すべてシュビーが合成**して 2 分（1:58）に仕上げる。

## 渡し方（最重要）

1. クリップを **`docs/ivrc-video/footage/` フォルダ**に置く。
2. ファイル名は**カット ID をそのまま**（拡張子は .mp4 / .mov どちらでも）:
   `COLD.mp4` `A2.mp4` `A3.mp4` `A4.mp4` `B0.mp4` `B1.mp4` `A5.mp4` `B4.mp4`（任意 `B3.mp4` `B2_latency.mp4`）
   - 名前を付けるのが面倒なら 1 フォルダにまとめて「どれがどれか」一言メモでくれれば私がリネームする。
3. 各クリップで**使ってほしいベスト区間（例「3〜14秒」）**をメモしてくれると助かる（無ければ私が選ぶ）。
4. **1 本ずつでもOK**。届いた分から順に仮の黒板を実映像へ差し替えて、その都度 2 分通しを再書き出しする。

## 全クリップ共通の技術仕様

- **横向き（16:9）**、できれば 1920×1080 / 30fps。スマホは横持ち。
- **三脚 or 固定**で手ブレなし。CCTV パートは“動かない画角”が命。
- 各クリップの**頭と尻に 2〜3 秒の静止**を残す（私がトリミングする「のり代」）。
- **音声は不要**（こちらで無音化。音楽は別工程）。同録があっても私は無視するので気にしないで。
- HMD 内映像（Quest 録画）は**アスペクトが正方形等でも可**。私が 16:9 にレターボックス/トリミングする。**ガーディアン枠・システム UI を映さない**ことだけ守る。
- mp4(H.264) 推奨。Quest 標準キャプチャ/キャスト録画でOK。

| **A4.mp4** | A4 | 8s | **20秒** | **ゾーン境界を跨いで歩く**と、見ている固定カメラが別画角へ自動切替。境界を 2 回またぐ（各 2〜3 秒静止してから移動） | Quest 画面録画 |
| **B0.mp4** | B0 | 10s | **20秒** | 現実の引きの全景：**三脚スマホ 3 台 ＋ Quest3 を被った体験者**が同じ部屋にいる。スマホが「カメラ源」と分かる画角 | 外部カメラ（三脚） |
| **B1.mp4** | B1 ★決め画 | 16s | **30〜40秒 ×2-3テイク** | **ワンショット証拠**：1 画角の中に〈中央→右→左と歩く体験者〉と〈HMD 映像を映した PC モニタ/キャスト画面〉を**同時に**収める。歩いて境界を跨ぐとモニタのカメラが自動切替（テイク内で 2 回）。床のゾーン境界に養生テープ | 外部カメラ（三脚）＋ PC に HMD 映像をキャスト |

> **撮影優先順位**: B1（決め画）→ A3（恐怖の核）→ B0（全景）→ A2 → A4 → COLD。時間が無ければこの順で確保。

## 背景透過オーバーレイ（あなたが背面に映像を敷いて完成させる用）

**`docs/ivrc-video/draft/ivrc-overlay.mov`**（qtrle / アルファ付き・約 21MB・1:58）を用意済み。
中身は **字幕 ＋ CCTV UI（四隅枠・REC・カメラ番号・タイムスタンプ）＋ 実機/イメージ バッジ ＋ 原理図/エンドカード/タイトル文字**。背景は完全透過（〔仮〕説明ボックス・PART 表示・進捗バーは無し）。

使い方（DaVinci Resolve / Premiere 等）:
1. 下のトラックに**あなたの実機映像**を、上のトラックに `ivrc-overlay.mov` を置く（尺は同じ 1:58）。
2. 各カットの区間（[storyboard.md](storyboard.md) の秒数）に対応する映像を背面に並べるだけで、字幕・UI が乗った完成形になる。
3. 図（B2 1:20–1:33）・エンドカード（B5 1:48–1:58）・タイトル背景は映像不要 → 背面は黒でよい。
4. 音楽は `audio/bed.*` を一番下のトラックに。
- 作り直しが要れば `python build_overlay.py` で再書き出し（台本やバッジを変えたら自動反映）。

## 完成までの流れ

1. あなたが上の必須 6 本（＋AI 2 本）を `footage/` に置く（1 本ずつでも可）。
2. シュビーが `build_final.py` を回す → 黒板の仮カットが実映像に置き換わり、UI・字幕・尺を合成して **`ivrc-final-v1.mp4`（1:58）** を書き出す。
3. 通しで確認 → 字幕尺・カットの間（ま）・トリミング位置を微調整して再書き出し（何度でも）。
4. **音楽・SFX（mux はシュビーがやる）**: ffmpeg 導入済み（`imageio-ffmpeg`・ユーザー空間）。あなたは [music-sfx.md](music-sfx.md) の素材を**1 本のミックス音声にして渡すだけ**。
   - 置き場所: `docs/ivrc-video/draft/audio/` に **`bed.wav`（または bed.mp3）** で置く（長さは 118 秒前後・足りなければ私がループ/トリム/フェード）。
   - 私が `finalize.py` を回す → 映像に mux して **`ivrc-final.mp4`（H.264＋AAC）** を書き出す。`audio/bed.*` があればそれを優先、無ければ**仮スクラッチ音**（合成・著作権フリー・タイミング確認用）で出す。
   - 凝った同期（カット頭に効果音をピタ当て等）が要るなら DaVinci で重ねる選択肢も残す。まずは bed 1 本で十分。
